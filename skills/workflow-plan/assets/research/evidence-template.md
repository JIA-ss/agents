## 证据索引格式（汇总用）

在 research.md 的附录中汇总所有证据：

```markdown
## 附录：证据索引

| ID | 时间 | 阶段 | 类型 | 状态 | 关键发现 | 来源 |
|----|------|------|------|------|----------|------|
| E-1 | 2026-01-16T10:00 | 1-overview | web-search | success | {发现} | {来源} |
| E-2 | 2026-01-16T10:05 | 2-current-state | code-analysis | success | {发现} | {路径} |
| E-3 | 2026-01-16T10:10 | 3-analysis | web-search | failed | - | - |
```

---

## 结论引用格式

在 research.md 的结论中引用证据：

```markdown
根据调研 [E-1][E-3]，推荐采用方案 A，理由如下...

基于代码分析 [E-2]，现有系统已有类似实现...
```

## 失败记录示例

当搜索/分析失败时，仍需记录：

```markdown
## 证据记录 #3

### 基本信息

| 字段 | 值 |
|------|-----|
| **ID** | E-3 |
| **时间** | 2026-01-16T10:10:00Z |
| **阶段** | 3-analysis |
| **类型** | web-search |
| **状态** | failed |

### 搜索/分析内容

**查询/操作**:
```
{搜索关键词}
```

### 关键发现

无（搜索失败）

### 备注

搜索失败原因：网络超时 / 无相关结果 / 访问被拒绝

后续处理：更换关键词重试 / 使用备用来源 / 跳过此证据点
```
