## Evolution Protocol

<!-- 此章节是 AI 的行为指令，使用时无需删除 -->

本 Skill 支持自动进化。每次使用后根据执行效果自动改进：

- **Skill 指导正确** → 无需操作
- **Skill 缺少关键信息**（你在执行中自行补充了知识）→ 将补充内容回写到对应章节（如 Core Rules / Common Issues / Best Practices）
- **Skill 有误导**（导致走弯路或错误）→ 修正错误内容，在 `.evolution/changelog.md` 追加记录：`| {日期} | feedback | {修正描述} |`
- **发现未覆盖的场景** → 补充到 Common Issues 或 Best Practices
- **用户纠正了匹配/理解** → 更新 frontmatter description 的触发词或修正 Core Rules

### Evolution 目录

```
.evolution/
├── changelog.md    # 变更和反馈记录（必须）
└── config.yaml     # 阈值配置（可选）
```

### Changelog 格式

```markdown
# Evolution Changelog
| 日期 | 类型 | 描述 |
|------|------|------|
| {日期} | sync | {Sync 变更描述} |
| {日期} | feedback | {执行反馈描述} |
| {日期} | fix | {修正描述} |
```
