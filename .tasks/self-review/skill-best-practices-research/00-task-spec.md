# Task Specification: Claude Code Skill Best Practices Research

## Metadata
- **Task ID**: skill-best-practices-research
- **Created**: 2026-01-12
- **Type**: Research

## Objective

调研 Claude Code Skill 的最佳实践，主要包括三个方面：
1. 如何生成 skill 最符合官方的设计
2. Skill 需要保证哪些规范，有哪些限制（字数、格式、资源等）
3. 如何设计才能让 skill 能够自我迭代，在工作中不断完善自己

## Scope

- Official Anthropic documentation
- Agent Skills specification (agentskills.io)
- GitHub anthropics/skills repository
- Best practices guide from platform.claude.com
- Local skill implementations analysis

## Success Criteria

- Comprehensive documentation of official skill specifications
- Clear constraints and limitations summary
- Actionable self-iteration design patterns
